<h1>Project Overview</h1>
The To-Do List Application is a command-line interface (CLI) tool designed to help users manage their daily tasks efficiently. It provides a simple yet effective environment for adding, viewing, and removing tasks in real-time.

<h1>Key Features</h1>
<h3>Dynamic Task Management:</h3> Users can add new tasks to their list and see them updated immediately.

<h3>Item Removal:</h3> Includes functionality to remove completed or unnecessary tasks from the current list.

<h3>Interactive CLI:</h3> Uses a numbered menu system for intuitive navigation and user input.

<h3>Session Persistence:</h3> Displays the current state of the list after every action so the user never loses track of their progress.

<h1>Technologies Used</h1>
<h3>Python:</h3> The core logic is built using Python, focusing on efficient data handling.

<h3>Control Flow:</h3> Utilizes loops and conditional statements to manage the application's lifecycle and user choices.

<h3>Data Structures:</h3> Employs Python lists to store and manipulate task data dynamically.
